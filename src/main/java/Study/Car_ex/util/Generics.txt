 Generics를 이용한 Collection API 사용
 Vector Stack Queue
 1. 자바에서 제공하는 다양한 컬렉션들은 다른 타입의 Object 들을 저장하고 관리하는 기능을 제공함.
 2. 하지만 실제 프로그램을 개발할 때는 다른 타입의 데이터를 저장할 일이 거의 없음.
 3. 오히려 다른 타입의 데이터들을 컬렉션에 저장함으로써 문제가 발생되는 경우도 있음.
 그래서 나온 것이 Generics

 int와 double을 동시에 가지는 Vector의 경우에 값을 다 더하는 형태를 지닌다면,
 두 형이 맞지 않아 ClassCastException이 발생하는 경우가 있음.


 Generics : 자바에서 제공하는 컬렉션 객체가 특정 타입의 데이터만 저장하고 사용할 수 있도록 지원함.
 - 특징 : 컬렉션에 저장할 객체의 타입을 제한해서 사용함.
 - 장점 :
 1. 타입의 안정성 제공
 2. 타입 체크와 형변환 과정 생략
 3. 코드의 간결화
 - 제네릭스를 사용하기 위한 문법
 컬렉션<데이터 타입> 변수이름 = new 컬렉션<데이터 타입>(); // 저장해서 사용할 데이터 타입 지정.
 ex) Vector<Integer> list = new Vector<Integer>();
